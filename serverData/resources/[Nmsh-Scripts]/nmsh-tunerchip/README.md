## Features
- Vehicle tuning with Tunerchip (Item name: tunerlaptop)
- Usable NOS for vehicles (Item name: nitrous)

## Installation
### Manual
- Download the script and put it in the `[qb]` directory.
- Add the following code to your server.cfg/resouces.cfg
```
ensure qb-core
ensure qb-hud
ensure qb-tunerchip
ensure vstancer
```
